# スーパーこうかとん

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
主人公キャラクターのこうかとんが走ってジャンプしてゴールを目指すゲーム

## ゲームの遊び方
* 矢印キーでこうかとんを操作し，スペースキー押下によりこうかとんがジャンプする
* ゴール地点に到達したらゲーム終了

## ゲームの実装
### 共通基本機能
* 背景画像と主人公キャラクターの描画

### 担当追加機能
* jump機能（担当：村上）：SPACEキーを押したら、こうかとんがジャンプし地面に重力で戻ってくる機能
