# スーパーこうかとん

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
主人公キャラクターこうかとんが走ってジャンプしてゴールを目指すゲーム

## ゲームの遊び方
* 矢印キーでこうかとんを操作し、ジャンプボタンでジャンプする
* ゴール地点に触れたらゴールになる

## ゲームの実装
### 共通基本機能
* 背景画像と主人公キャラクターの描画、こうかとんの移動

### 担当追加機能
* ステージの床を実装（担当：金井）：床に関する機能
